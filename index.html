Switch Region Checker

This tool lets you check which versions of Switch games are sold in different regions. This is useful for knowing if you can, for example, use your European copy of a game with the USA version of the DLC, which should work if both regions use the same version of the game. You can also check your game's code and version in the back of the box, just above the barcode.
